# Missing Assets

This directory contains placeholder files. You need to replace them with the actual assets from your template.

## Fonts

The template requires these fonts:
- Source Sans Pro (regular, 700)
- Montserrat (500, 600, 700)
- Playfair Display (regular, italic, 900)

**Options:**
1. Download from Google Fonts: https://fonts.google.com/
2. Use CDN links in your HTML instead of local files
3. Get the original font files from your template package
